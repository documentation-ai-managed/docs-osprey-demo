## Prerequisites

<Callout kind="info" title="Before You Begin">
Ensure you have:
- Osprey Tilt String (OTS) or In-Place Extensometer (IPX) kit
- Power source (battery or mains adapter)
- Laptop with USB port or Ethernet connection
- Latest Osprey software tools downloaded from the support page
- Site survey confirming mounting locations
</Callout>

Verify your environment meets these requirements to avoid setup delays.

Unbox your Osprey instrumentation carefully to prevent damage to sensors.

## Installation Steps

Follow these steps to install your Osprey device on site.

<Steps>
  <Step title="Prepare Site" icon="map">
    Select a stable mounting point free from excessive vibration. Clean the surface and mark positions using a level.
  </Step>
  <Step title="Mount Sensors" icon="settings">
    Attach brackets securely. For OTS, lay the string along the asset; for IPX, install anchors at depth intervals.
  </Step>
  <Step title="Connect Power and Data" icon="plug">
    Wire to power source and connect to data logger. Use weatherproof enclosures outdoors.
  </Step>
  <Step title="Power On" icon="zap">
    Switch on and check indicator lights for initial status.
  </Step>
</Steps>

<Tabs>
  <Tab title="Tilt String (OTS)" icon="activity">
    Embed in concrete or trench-lay for railways. Space sensors 1-2m apart based on needs.
  </Tab>
  <Tab title="Extensometer (IPX)" icon="ruler">
    Bore holes, insert anchors, grout in place. Allow 24 hours curing before tensioning.
  </Tab>
</Tabs>

## First Data Reading and Verification

Connect your device to the Osprey Dashboard via USB or Ethernet. Use the API to fetch initial readings.

<CodeGroup tabs="JavaScript,Python">
  ```javascript
  const OspreyAPI = require('osprey-api');

  async function readData() {
    const client = new OspreyAPI('https://api.example.com', 'YOUR_API_KEY');
    const data = await client.getTiltReadings('device-123');
    console.log(data.tilts); // Array of tilt values in degrees
  }

  readData();
  ```
  ```python
  import requests

  API_URL = 'https://api.example.com'
  API_KEY = 'YOUR_API_KEY'
  DEVICE_ID = 'device-123'

  response = requests.get(
      f'{API_URL}/tilts/{DEVICE_ID}',
      headers={'Authorization': f'Bearer {API_KEY}'}
  )
  tilts = response.json()['tilts']
  print(tilts)  # List of tilt measurements
  ```
</CodeGroup>

Verify readings show stable baselines (e.g., `<0.1>` degrees tilt). Log any anomalies.

<Request tabs="cURL,JavaScript" show-lines="true">
  ```bash
  curl -X GET https://api.example.com/tilts/device-123 \
    -H "Authorization: Bearer YOUR_API_KEY"
  ```
  ```javascript
  fetch('https://api.example.com/tilts/device-123', {
    headers: { 'Authorization': 'Bearer YOUR_API_KEY' }
  }).then(res => res.json()).then(console.log);
  ```
</Request>

<Response tabs="200">
  ```json
  {
    "device": "device-123",
    "tilts": [0.02, -0.01, 0.05],
    "timestamp": "2024-01-15T10:30:00Z"
  }
  ```
</Response>

## Safety Considerations

<Callout kind="alert" title="Safety First">
- Wear PPE: helmet, gloves, safety glasses
- De-energize site power before wiring
- Avoid overhead work without spotters
- Handle grout chemicals with careâ€”follow SDS
- Never exceed sensor vibration thresholds
</Callout>

<Expandable title="Common Troubleshooting" default-open="false">
- No data: Check connections with multimeter
- Erratic readings: Realign sensors
- Power issues: Verify voltage `>12V`
</Expandable>
