## Products

<Columns cols="2">
  <Card title="Osprey Tilt String (OTS)" href="/ots/overview" icon="activity" horizontal="false">
    High-precision tilt measurements across up to 150 positions on a single cable. Ideal for vertical assets like towers and linear structures like pipelines.
  </Card>

  <Card title="In-Place Extensometer (IPX)" href="/ipx/overview" icon="ruler" horizontal="false">
    Precise strain monitoring with robust, low-profile design for embedded or exposed use.
  </Card>

  <Card title="Automatic Settlement Profiler (ASP)" href="/asp/overview" icon="droplets" horizontal="false">
    Distributed settlement measurements along a nominally horizontal profile using hydrostatic principle.
  </Card>

  <Card title="General Products" href="/general/overview" icon="cog" horizontal="false">

  </Card>
