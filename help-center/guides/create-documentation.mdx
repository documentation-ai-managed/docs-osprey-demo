## Quick Guide

<Steps>
  <Step title="Open Your Project">
    Select your project from the dashboard.
  </Step>
  <Step title="Add a New Page">
    Click "+ New Page" and enter a title.
  </Step>
  <Step title="Write Content">
    Use the editor to add text, lists, or code blocks. Save changes.
  </Step>
</Steps>

## Next Steps
